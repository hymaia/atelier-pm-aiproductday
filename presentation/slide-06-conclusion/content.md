## Mapping concepts

| Concept Claude Code | Ce que c'est | Analogie PM |
|--------------------|--------------|-------------|
| `CLAUDE.md` | Instructions permanentes pour Claude | Votre brief d'onboarding |
| `data/` | Vos fichiers source | Votre base de connaissance |
| `indexer.py` | Pipeline qui prépare les données | Votre ETL maison |
| `loaders/` | Connecteurs vers vos sources | Vos intégrations |
| `skills/` | Commandes métier personnalisées | Vos macros Excel, mais en IA |
| `.claude/` | Configuration locale du projet | Votre workspace settings |

## Ce que vous pouvez faire maintenant

- **Brancher vos vraies données** : calendrier, tickets, interviews
- **Créer vos propres skills** : tout ce que vous faites de répétitif
- **Partager le repo** avec votre équipe — chacun son contexte, même OS
