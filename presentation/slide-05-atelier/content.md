## Les 3 règles

1. **Ouvrez Claude Code** dans le dossier de votre module (`cd pm-[module] && claude`)
2. **Dites à Claude ce que vous voulez faire** — soyez précis sur votre objectif
3. **Committez vos changements** — on fera un tour de table à la fin

## Ressources

- Repo : `git clone git@github.com:hymaia/atelier-pm-aiproductday.git`
- Docs Claude Code : `claude --help`
- En cas de blocage : levez la main, on vient vous voir

## Timer

⏱️ **30 minutes**
