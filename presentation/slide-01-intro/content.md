## Pourquoi vous êtes là

Claude Code n'est pas qu'un outil de code.
C'est un **environnement de travail programmable** pour n'importe quel métier.

Ce projet = un **OS IA pour Product Manager** :
un assistant qui connaît votre contexte, vos clients, votre agenda.

**Ce matin : découvrir → tester → personnaliser.**

## Prérequis

- [ ] Claude Code installé : `npm install -g @anthropic-ai/claude-code`
- [ ] Repo cloné : `git clone git@github.com:hymaia/atelier-pm-aiproductday.git`
- [ ] Clé API Anthropic : `export ANTHROPIC_API_KEY=sk-ant-...`
- [ ] Lancer Claude Code : `claude` dans le dossier du projet
